Prompt Library

Selected prompts and templates I use for design, code, and strategy. Copy, adapt, and ship.

Product Spec (Concise)
You are a senior PM. Create a concise 1-pager with:
- Problem
- Goals & non-goals
- Users & use cases
- KPI(s)
- Solution outline (MVP)
- Risks & open questions
Return as structured Markdown.
Code Review (Actionable)
Review the diff for correctness, readability, and risks.
Call out:
- Naming and clarity
- Edge cases and error handling
- Test coverage suggestions
Return a short checklist with proposed edits.
Design Crit (Objective)
Evaluate the design against:
- Hierarchy and spacing
- Contrast and color accessibility
- Copy tone and brevity
- Mobile responsiveness
Give prioritized, actionable feedback.
Meeting Notes → Actions
Given raw meeting notes, extract:
- Decisions
- Action items (owner, due date)
- Risks/blockers
Return in a table, then a concise summary paragraph.
Job Story Generator
Generate X job stories using the format:
When <situation>, I want <motivation> so I can <expected outcome>.
Vary audiences and constraints; keep each to one line.
