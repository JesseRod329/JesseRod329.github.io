Admin Page Test

This page tests the admin functionality:

Test Steps:

  1. Open browser dev tools (F12)
  2. Go to Console tab
  3. Navigate to admin.html
  4. Check console logs for AdminSystem.init() messages
  5. If not authenticated, you should see "Access Denied"
  6. If authenticated as admin, you should see the admin interface

Expected Console Output:

AdminSystem.init() called
Current URL: http://localhost:8000/admin.html
AppState.isAuthenticated: true/false
AppState.currentUser: {email: "..."}
User email: jesse.rodriguez89@gmail.com
Allowed admins: ["jesse.rodriguez89@gmail.com"]
User is admin, showing admin content
AdminSystem.showAdminContent() called
adminContent element: 

Showed admin content
AdminSystem.loadEvents() called
Events loaded: [...]
Events populated in dropdown
AdminSystem.bindEventListeners() called
Event select element found, adding listener

Troubleshooting:

  • If you see "adminContent element not found" - there's an HTML issue
  • If you see "User not authenticated" - sign in first
  • If you see "User not in admin list" - your email isn't in ALLOWED_ADMINS
  • If you see "Event select element not found" - there's a DOM issue
