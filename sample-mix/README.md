# Sample Mix

A professional audio chopping and MIDI pad player tool for music producers. Upload any song, automatically detect beats and chop it into samples, then play them on an intuitive MIDI pad interface.

## Features

- **Audio Upload**: Support for MP3, WAV, OGG, and M4A formats
- **Automatic Beat Detection**: Intelligent algorithm detects transients and chops audio automatically
- **Manual Slice Editing**: Click on the waveform to add, drag, or remove slice markers
- **MIDI Pad Interface**: 32-pad grid with keyboard shortcuts (Q-P, A-L, Z-M, 1-0)
- **Waveform Visualization**: Real-time waveform display with slice markers
- **Master Volume Control**: Adjustable volume for all pads
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Dark Mode Support**: Automatically adapts to your system preferences

## Usage

1. **Upload Audio**: Drag and drop an audio file or click to browse
2. **Auto Chop**: Click "Auto Chop" to automatically detect beats and create slices
3. **Adjust Settings**: Use sensitivity and minimum slice length sliders to fine-tune chopping
4. **Play Samples**: Click pads or use keyboard shortcuts to trigger samples
5. **Manual Editing**: Click on the waveform to add slice markers, drag to move them, or double-click to remove

## Keyboard Shortcuts

- **Q-P**: Pads 1-10
- **A-L**: Pads 11-19
- **Z-M**: Pads 20-26
- **1-0**: Pads 27-36

## Technical Details

- Built with Web Audio API for high-performance audio processing
- Canvas-based waveform visualization
- Real-time audio buffer slicing and playback
- Optimized for low latency and smooth performance

## Browser Support

Works best in modern browsers with Web Audio API support:
- Chrome/Edge (recommended)
- Firefox
- Safari
- Opera

## Privacy

All audio processing happens locally in your browser. No audio files are uploaded to any server.
