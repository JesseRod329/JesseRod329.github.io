## Capitol Trades Dataset

This folder holds the raw and derived datasets scraped from [Capitol Trades](https://www.capitoltrades.com/trades).

### Available files

- `trades.jsonl` – newline-delimited JSON generated by `npm run scrape:capitoltrades`
- `trades.csv` – flattened CSV export generated by `npm run trades:csv`
- `summary.json` – metadata about the most recent scrape (record counts, scrape timestamp, etc.)

### Updating the data

1. Re-scrape the source site (takes ~6 minutes for ~2.9k pages):

   ```bash
   npm run scrape:capitoltrades
   ```

2. Regenerate the CSV export from the JSONL file:

   ```bash
   npm run trades:csv
   ```

Both scripts accept optional `--input` and `--output` flags if you want to place the files elsewhere. See the header comments in each script for the exact CLI usage.
