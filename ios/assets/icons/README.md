# PWA Icons Directory

This directory contains app icons for the iOS Portfolio PWA.

## Required Icons

- `apple-touch-icon.png` (180x180) - iOS home screen icon
- `favicon-32x32.png` (32x32) - Desktop favicon
- `favicon-16x16.png` (16x16) - Small favicon
